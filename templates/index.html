Job Orchestrator API

A lightweight Flask microservice for creating, tracking, and updating background jobs and tasks.

Status: Running

Available Endpoints

POST   /tasks -- adds a new job
GET    /tasks -- retrieves all jobs
GET    /tasks/<task_id> -- retrieves a specific job, can filtered by status
PATCH /tasks/<task_id> -- updates a specific job
DELETE /tasks/<task_id> -- deletes a specific job
GET    /health -- health check endpoint

Quick Example

        curl -X POST http://localhost:8080/tasks \\
         -H "Content-Type: application/json" \\
         -d '{"name": "Generate report", "job_type": "reports", "created_by": "user123", "payload": {"report_id": 123}}'

Notes

  • Backed by SQLite
  • Designed as an API-only microservice
  • Payloads stored as text for simplicity
  • Health endpoint intended for monitoring
